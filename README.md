# D7sysinfo

Domain 7 Solutions system information.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'd7sysinfo'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install d7sysinfo

## Usage

d7sysinfo --json will output server information

## Prerequisites
 * uri
 * net
 * rexml
